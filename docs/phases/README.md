## 読む判断

* 現在の実装対象を見る場合は [Phase 7+ 再編計画](phase-7-plus-reorg.md) の Phase 8/9 section を先に読む。
* Phase 10 の完了条件、採用条件、review evidence、残る listening gap を見る場合は [Phase 10](phase-10.md) を読む。
* 実装対象の Phase が分かっている場合は、その Phase だけ読む。
